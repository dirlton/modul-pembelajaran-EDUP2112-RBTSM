# 📚 Apa Itu Projek Ini?

Projek ini ialah sebuah laman web pembelajaran interaktif yang dibina menggunakan:

* HTML
* CSS
* JavaScript

Tujuan projek ini adalah untuk:

* membantu proses pembelajaran
* menjadikan pembelajaran lebih interaktif
* menggunakan elemen gamifikasi
* mengelakkan pembelajaran yang membosankan

---

# 🚀 Cara Menjalankan Projek

## LANGKAH 1 — Download VS Code

Download VS Code:

[https://code.visualstudio.com/](https://code.visualstudio.com/)

Install seperti biasa.

---

## LANGKAH 2 — Download Git

Download Git:

[https://git-scm.com/downloads](https://git-scm.com/downloads)

Semasa installation:

Pilih:

```plaintext
Git from the command line and also from 3rd-party software
```

---

# LANGKAH 3 — Clone Repository

Buka terminal dalam VS Code.

Masukkan command:

```bash
git clone https://github.com/dirlton/modul-pembelajaran-EDUP2112-RBTSM.git
```

---

# LANGKAH 4 — Buka Folder Projek

```bash
cd modul-pembelajaran-EDUP2112-RBTSM
```

---

# LANGKAH 5 — Jalankan Projek

Buka:

```plaintext
index.html
```

menggunakan browser.

ATAU gunakan extension:

## Live Server

Dalam VS Code:

1. Pergi ke Extensions
2. Cari:

```plaintext
Live Server
```

3. Install
4. Klik kanan pada `index.html`
5. Tekan:

```plaintext
Open with Live Server
```
